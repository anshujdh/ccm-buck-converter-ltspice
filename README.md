\# CCM Buck Converter (LTspice)



An open-loop Continuous Conduction Mode (CCM) buck converter designed and simulated in LTspice.



\## Specifications



\- Input Voltage: 50 V

\- Output Voltage: 20 V

\- Duty Cycle: 40%

\- Switching Frequency: 20 kHz

\- Inductor: 400 µH

\- Load Resistance: 10 Ω
